##  Specious Sites
GitHub repository for websites utilized to document the popularity and influence of different unreliable news websites. This dataset consists of 1,334 unreliable news websites. This GitHub repository will be updated with the code and weights of the model utilized in our paper.

For additional details about the collection method and analysis of these websites, see our paper/analysis here: https://www.hanshanley.com/files/Specious_Sites.pdf

## Citing the paper
